🔍 Sprint 03: Ransomware Incident Analysis & Strategic Remediation
Objective: To conduct a post-incident forensic review for the Rigel Cybernetic Institute (RCI) following a ransomware attack. This project involved identifying root causes, clustering systemic vulnerabilities, and proposing prioritized solutions mapped to the NIST Cybersecurity Framework (CSF).

🛠️ The Analysis Lifecycle
Evidence Mining: Critically reviewed the RCI casefile to enumerate specific security failures and operational gaps during the incident.

Root Cause Analysis (RCA): Performed a deep dive into identified problems to move past symptoms and address the underlying technical and administrative weaknesses.

Strategic Clustering: Analyzed and consolidated duplicate root causes into high-impact "problem clusters," streamlining the remediation roadmap.

NIST CSF Mapping: Categorized all findings and proposed solutions across the five NIST core functions: Identify, Protect, Detect, Respond, and Recover.

Risk-Based Prioritization: Ranked security recommendations based on RCI’s specific business priorities, balancing maximum risk reduction with budgetary and operational constraints.

📄 Project Deliverables
RCI_Cybersecurity_Analysis_Report.pdf: A formal, client-facing executive report documenting the incident, root causes, and a prioritized action plan for hardening the organization’s posture.

Root_Cause_Clustering_Matrix.csv: (If applicable) A breakdown of the raw data used to categorize and prioritize the findings.
